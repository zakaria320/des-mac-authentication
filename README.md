
# DES-Based MAC Authentication Tool

A Python security tool that generates Message Authentication Codes using DES encryption and demonstrates brute-force attack vulnerabilities.

 Features
- MAC generation using DES in ECB mode
- Pattern-based brute-force attack (72,800 combinations)
- Exhaustive key search for variable-length keys
- Real-time progress tracking
- Automated key recovery system

 Technologies Used
- Python
- PyCrypto (DES encryption)

 Installation
```bash
pip install -r requirements.txt
```

 How to Run
```bash
python [your-filename].py
```

 What This Demonstrates
- DES encryption implementation
- Vulnerabilities in short encryption keys
- Brute-force attack algorithms
- Importance of strong key selection
